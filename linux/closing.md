# Linux — Advancement Criteria

A student is ready to advance to Docker when they can:

1. Navigate the filesystem using `pwd`, `ls`, `cd`
2. Explain absolute vs relative paths
3. Read and modify file permissions with `chmod` and `chown`

## Knowledge Check Questions

- What does `ls -la` show that `ls` doesn't?
- What is the difference between `/home/user` and `~`?
- What does `chmod 644` mean for owner, group, and others?
