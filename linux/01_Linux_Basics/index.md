# Linux Basics

Linux is an open-source operating system kernel. Most servers run Linux.

## Key Concepts

- **Kernel** — the core of the OS, manages hardware resources
- **Shell** — the command-line interface (e.g. `bash`, `zsh`)
- **Prompt** — the text before your cursor; usually shows your username and current directory

## Essential Commands

```bash
pwd       # print current directory
ls        # list files
ls -la    # list files with details and hidden files
cd /etc   # change to /etc directory
cd ~      # go to your home directory
cd ..     # go up one level
man ls    # read the manual for any command
```
