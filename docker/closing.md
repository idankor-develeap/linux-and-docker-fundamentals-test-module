# Docker — Advancement Criteria

A student is ready to advance when they can:

1. Explain the difference between an image and a container
2. Pull an image, run a container, and remove it
3. Use volumes to persist data across container restarts

## Knowledge Check Questions

- What happens to data inside a container when you run `docker rm`?
- What is the difference between `docker stop` and `docker rm`?
- How do you see logs from a running container?
